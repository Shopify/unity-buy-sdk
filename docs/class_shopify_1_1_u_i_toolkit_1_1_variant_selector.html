Detailed Description

Converts the list of product variant and their respective selectable options that looks like: [{ "selectedOptions": [{ "name": "optionName1", "value": "value1" }, { "name": "optionName2", "value": "value1" }] }, { "selectedOptions": [{ "name": "optionName1", "value": "value2" }, { "name": "optionName2", "value": "value2" }] }] into a map that looks like: { "optionName1": ["value1", "value2", ...], "optionName2": ["value1", "value2", ...], }
